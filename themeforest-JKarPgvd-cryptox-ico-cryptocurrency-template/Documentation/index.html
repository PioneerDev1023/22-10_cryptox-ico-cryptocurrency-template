Mailchimp integration

Be sure that the form created in your mailchimp dashboard is asking only the email address and not the name, surname or something else.

  • 1. Click your profile name to open the Account Panel and choose Account.
  • 2. Click the Extras drop-down menu and choose API keys.
  • 3. Copy an existing API key or click the Create A Key button.
  • 4. Be careful that your list asks just the email address, Lists > Signup forms > General forms > Build it. See the screenshot below.
  • 5. Navigate to the Lists page on the top left of your dashboard.
  • 6. Click the list's drop down menu and select Settings.
  • 7. Click on List name & defaults to find the unique List ID for the list, see the screenshot below.
  • 8. Open the file subscribe.php findable in the folder php and fill up the API_KEY and LIST_ID fields.
    • $mailchimp_api_key = 'XXXXXXXXXXXXXXXXXXXXXXXXXXXXX-XXXX'; // enter your MailChimp API Key
$mailchimp_list_id = 'XXXXXXXXXX'; // enter your MailChimp List ID
